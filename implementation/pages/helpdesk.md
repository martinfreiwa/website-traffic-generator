# Help Desk Implementation Plan

**Component**: `HelpDesk.tsx`
**Route**: `/helpdesk`

## Current Features
*   Knowledge base article listing.
*   Search functionality.
*   Contact support form/link.

## Planned Improvements
*   [ ] **Categories**: Group articles by Topic (Billing, Technical).
*   [ ] **Most Popular**: "Top 5 Articles" widget based on views.
*   [ ] **Related Articles**: "Users who read this also read..." at bottom of post.
*   [ ] **Video Tutorials**: Embed Loom/YouTube walk-throughs in articles.
*   [ ] **Feedback Widget**: "Was this helpful? Yes/No" at bottom.
*   [ ] **Glossary**: A-Z dictionary of platform terms.
*   [ ] **Breadcrumbs**: Home > Help > Billing > Invoice.
*   [ ] **Search Highlight**: Highlight the matching keyword in search results.
*   [ ] **Live Search**: Dropdown results appear as you type.
*   [ ] **Contact Form**: "Can't find answer?" form at bottom.
*   [ ] **Ticket Status**: "Check your ticket status" input field.
*   [ ] **System Status**: "All Systems Operational" banner.
*   [ ] **Last Updated**: "Article updated 2 days ago" badge.
*   [ ] **Author Profile**: "Written by [Support Agent]" with photo.
*   [ ] **Print Button**: "Print this guide" functionality.
*   [ ] **PDF Download**: "Download Manual" button.
*   [ ] **Table of Contents**: Internal links for long guides.
*   [ ] **Code Snippets**: "Copy to clipboard" for API examples.
*   [ ] **Interactive Troubleshooter**: Step-by-step "wizard" to solve problems.
*   [ ] **Community Link**: "Ask the community" button.
*   [ ] **Article Rating**: 1-5 star rating interaction.
*   [ ] **Tag Cloud**: Filter by tags (e.g. #error, #payment).
*   [ ] **Changelog Link**: "See what's new" link.
*   [ ] **Reading Time**: "3 min read" indicator.
*   [ ] **Next/Prev**: Navigation to next article in series.
*   [ ] **Mobile Optimization**: Collapsible sections on phone.
*   [ ] **Warning Boxes**: "Note: This feature is Pro only" alert boxes.
*   [ ] **GIF Explainer**: Animated GIFs showing UI interactions.
*   [ ] **Version Select**: "Viewing instructions for V2" dropdown.
*   [ ] **Language**: Toggle article language.
*   [ ] **Sticky Sidebar**: Navigation menu stays visible on scroll.
*   [ ] **Lazy Load**: Images load only when scrolled to.
*   [ ] **404 Search**: If article not found, auto-search usage.
*   [ ] **Schema Markup**: FAQPage schema for Google.
*   [ ] **Quick Links**: "Password Reset" | "Billing" buttons.
*   [ ] **Dark Mode**: Support docs in dark mode.
*   [ ] **Live Chat Trigger**: "Chat with us" button floating.
*   [ ] **Email Article**: "Email this to me" button.
*   [ ] **Keyboard Search**: Press `/` to focus search bar.
*   [ ] **Glossary Tooltip**: Hover over technical terms.
*   [ ] **Deep Links**: Link to specific H2 headers.
*   [ ] **Bot Deflection**: "Before you contact us, check this" modal.
