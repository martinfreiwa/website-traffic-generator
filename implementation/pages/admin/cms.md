# Admin Content Management (CMS) Implementation Plan

**Components**: `AdminCMS.tsx`, `AdminBlog.tsx`, `AdminLegal.tsx`
**Routes**: `/admin/cms`, `/admin/blog`

## Purpose
Manage public-facing content without needing code deploys.

## Features
*   [ ] **Blog Editor**: Create/Edit posts.
*   [ ] **Page Editor**: Edit static pages (About, Terms).
*   [ ] **Media Library**: Upload images/files.

## Planned Improvements
*   [ ] **WYSIWYG Editor**: Rich text editor (tiptap/quill) for content.
*   [ ] **SEO Meta Tools**: Preview Google snippet while editing.
*   [ ] **Draft Mode**: Save without publishing.
*   [ ] **Scheduled Publish**: "Go live on [Date]".
*   [ ] **Version History**: Rollback to previous version of article.
*   [ ] **Category Manager**: Create/Edit blog categories.
*   [ ] **Author Management**: Create author profiles (Bio, Image).
*   [ ] **Featured Image**: Crop/Resize tool for post thumbnails.
*   [ ] **Tag Manager**: Manage tag cloud.
*   [ ] **Preview Mode**: "View as User" button before publishing.
*   [ ] **Slug Editor**: Custom URL per post.
*   [ ] **Markdown Support**: Toggle between Visual and Markdown mode.
*   [ ] **Import/Export**: Move content from WordPress/Ghost.
*   [ ] **Social Preview**: "How this looks on Twitter/Facebook".
*   [ ] **Table of Contents**: Auto-generate ToC from headers.
*   [ ] **Code Highlighting**: Support code blocks in articles.
*   [ ] **Sticky Post**: "Pin to top" toggle.
*   [ ] **Comment Moderation**: Approve/Delete comments (if enabled).
*   [ ] **Embed Support**: Easy YouTube/Twitter widget insertion.
*   [ ] **Legal Versioning**: "Update Terms" -> prompts user to re-accept.
*   [ ] **Asset Optimization**: Auto-compress uploaded images.
*   [ ] **FAQ Builder**: Drag-and-drop FAQ section editor.
*   [ ] **Testimonial Manager**: database of user quotes to display on site.
*   [ ] **Banner Manager**: Global "Alert Bar" text editor.
*   [ ] **Footer Editor**: Manage footer link columns.
*   [ ] **Menu Editor**: Reorder header navigation items.
*   [ ] **Redirect Manager**: Set up 301 redirects for old URLs.
*   [ ] **Email Templates**: HTML editor for system emails.
*   [ ] **Translation**: UI for managing multi-language content strings.
*   [ ] **Global Variables**: Define "Company Name" once, use everywhere.
*   [ ] **Form Builder**: Create custom contact forms.
*   [ ] **Popup Builder**: Create marketing popups/modals.
*   [ ] **A/B Tests**: Define variants for headlines.
*   [ ] **View Count**: manually adjust or view hit counters.
*   [ ] **RSS Feed**: Toggle/Config for blog RSS.
*   [ ] **Sitemap Exclusion**: Checkbox to hide page from sitemap.
*   [ ] **Canonical URL**: Custom override for SEO.
*   [ ] **Schema Markup**: JSON-LD editor per page.
*   [ ] **Access Control**: "Lock page" (password protected).
*   [ ] **Revision Compare**: Diff view of text changes.
*   [ ] **Broken Link Checker**: Scan content for dead links.
*   [ ] **Word Count**: "Time to read" calculator.
