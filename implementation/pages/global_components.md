# Global Components Implementation Plan

This file tracks shared components used across multiple pages.

## Core UI
*   `Sidebar.tsx`: Navigation menu for dashboard.
*   `Header.tsx`: Top bar for dashboard.
*   `ChatWidget.tsx`: Persistent support chat.
*   `Footer.tsx`: Public page footer.
*   `Button.tsx`: Reusable button (Primary, Secondary, Danger).
*   `Input.tsx`: Standard text input with validation.
*   `Modal.tsx`: Popup dialog wrapper.
*   `Toast.tsx`: Notification snackbar system.
*   `Card.tsx`: White container with shadow.
*   `Loader.tsx`: Spinner/Skeleton loading state.
