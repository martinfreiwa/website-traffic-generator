# Legal Pages Implementation Plan

**Component**: `Legal.tsx`
**Routes**: `/legal/*` (Privacy, Terms, Refund, etc.)

## Current Features
*   Terms of Service.
*   Privacy Policy.
*   Refund Policy.
*   Impressum.

## Planned Improvements
*   [ ] **Search in Page**: Input to search text within the legal doc.
*   [ ] **Contact DPO**: "Contact Data Protection Officer" email link.
*   [ ] **Version History**: Dropdown to view "Terms from 2023".
*   [ ] **Download PDF**: Button to download signed/official PDF.
*   [ ] **Print View**: CSS specifically for printing contracts.
*   [ ] **Last Updated**: Prominent date stamp at top.
*   [ ] **Table of Contents**: Quick links to sections (Credits, Liability, etc.).
*   [ ] **Language**: Toggle between English / German (Impressum requirement).
*   [ ] **Cookie Preferences**: Button to re-open cookie consent manager.
*   [ ] **Accept Button**: "I Agree" button (if used in onboarding).
*   [ ] **Highlight Changes**: "See what changed since last version".
*   [ ] **Jump to Section**: Smooth scroll to anchor links.
*   [ ] **Definition Tooltips**: Hover over "User" to see legal definition.
*   [ ] **FAQ Link**: "Have questions about this?" link to helpdesk.
*   [ ] **Email to Me**: Send copy of terms to user email.
*   [ ] **Summary Box**: "TL;DR" summary at top of complex sections.
*   [ ] **Embed Logic**: Component-ized for embedding in Signup modal.
*   [ ] **Sub-processor List**: Table of 3rd party data processors (GDPR).
*   [ ] **Opt-out Forms**: "Do Not Sell My Data" (CCPA) form.
*   [ ] **Request Data**: Form to request GDPR data export.
*   [ ] **Delete Account**: Instructions on how to cancel/delete.
*   [ ] **Address Map**: Embed map of company HQ (Impressum).
*   [ ] **VAT ID**: Display VAT/Tax ID clearly.
*   [ ] **Disclaimer**: "Not legal advice" footer.
*   [ ] **Accessibility**: ARIA labels for screen readers.
*   [ ] **Mobile Readability**: Large text/padding for phone reading.
*   [ ] **Deep Linking**: H2/H3 tags have ID for direct linking.
*   [ ] **Notion Sync**: Fetch legal text from Notion (CMS).
*   [ ] **Sticky Header**: "Terms of Service" title stays visible.
*   [ ] **Back to Home**: Clear exit navigation.
*   [ ] **Related Docs**: "See also: Privacy Policy" links.
*   [ ] **Signature Block**: (Optional) digital signature visual.
*   [ ] **Text Size**: A+/A- toggle.
*   [ ] **Dark Mode**: High contrast mode.
*   [ ] **Copy Text**: Button to select all text.
*   [ ] **Archive**: Link to "Archived Legal Docs" page.
*   [ ] **Jurisdiction**: "Governed by laws of [Country]" Highlight.
*   [ ] **Dispute Resolution**: Link to ODR platform (EU).
*   [ ] **Report Abuse**: Link to abuse report form.
*   [ ] **Age Gate**: "You must be 18+" confirmation.
