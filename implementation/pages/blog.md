# Blog Implementation Plan

**Component**: `Blog.tsx`
**Route**: `/blog`

## Current Features
*   List of blog posts.
*   Individual post view (likely).
*   SEO meta tags for articles.

## Planned Improvements
*   [ ] **Headless CMS**: Integrate with Contentful/Strapi/Sanity.
*   [ ] **Comments System**: Disqus or custom comment thread.
*   [ ] **Social Sharing**: Sticky side bar with Share buttons.
*   [ ] **Newsletter Signup**: "Subscribe for updates" mid-content box.
*   [ ] **Author Bio**: Detailed author card with social links.
*   [ ] **Related Posts**: "Read Next" grid at bottom.
*   [ ] **Table of Contents**: Auto-generated sticky ToC.
*   [ ] **Featured Post**: Large hero layout for newest/best article.
*   [ ] **Progress Bar**: Reading progress indicator at top of page.
*   [ ] **Category Filter**: Filter by "SEO", "Marketing", "Tech".
*   [ ] **Search**: Blog-specific search input.
*   [ ] **Archive**: Browse by Month/Year.
*   [ ] **Tag Cloud**: Visual cloud of popular tags.
*   [ ] **Estimated Read Time**: "5 min read".
*   [ ] **Series Navigation**: "Part 1 of 5" links.
*   [ ] **Lead Magnet**: "Download checklist" CTA in sidebar.
*   [ ] **Video Embed**: Support for YouTube/Vimeo in posts.
*   [ ] **Code Highlighter**: Prism.js syntax highlighting.
*   [ ] **Quote Block**: Styled blockquote for emphasis.
*   [ ] **Image Lightbox**: Click image to expand fullscreen.
*   [ ] **Lazy Loading**: Native lazy loading for post images.
*   [ ] **AMP Support**: Accelerated Mobile Pages version (optional).
*   [ ] **RSS Feed**: XML feed link.
*   [ ] **Guest Post**: "Write for us" link/form.
*   [ ] **Audio Player**: Text-to-speech player for article.
*   [ ] **Poll**: Embed interactive poll in article.
*   [ ] **Tweet Embed**: Native Twitter card styling.
*   [ ] **Sponsored Tag**: "Sponsored" label for paid posts.
*   [ ] **Update Date**: "Last updated on [Date]" (good for SEO).
*   [ ] **Editor's Pick**: Sidebar widget for curated top posts.
*   [ ] **Infinite Scroll**: Auto-load next post when reaching bottom.
*   [ ] **Font Size**: User control for text size.
*   [ ] **Dark Mode**: Blog-specific theme toggle.
*   [ ] **Print CSS**: Clean print layout.
*   [ ] **Schema.org**: Article schema JSON-LD.
*   [ ] **Open Graph**: Dynamic OG image generation with title.
*   [ ] **Breadcrumbs**: Home > Blog > Category > Post.
*   [ ] **Hover Effects**: subtle zoom on article card images.
*   [ ] **Pagination**: "Page 1 of 10" navigation.
*   [ ] **Sidebar Sticky**: Sidebar stays fixed on scroll.
*   [ ] **Mobile Table**: Horizontal scroll for data tables.
*   [ ] **CTA Injection**: Auto-insert "Buy Traffic" CTA after paragraph 3.
