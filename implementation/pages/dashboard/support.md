# Support Implementation Plan

**Component**: `Support.tsx`
**Route**: `/dashboard/support`

## Purpose
Direct channel for users to get help with their campaigns.

## Features
*   [ ] **Ticket System**: Create, view, and reply to support tickets.
*   [ ] **FAQ Link**: Quick access to help desk.
*   [ ] **Live Chat**: Integration with `ChatWidget`.

## Planned Improvements
*   [ ] **File Attachments**: Allow screenshots (drag & drop) in tickets.
*   [ ] **Ticket Rating**: Allow users to rate support quality after closure (1-5 stars).
*   [ ] **Instant Answers**: AI suggestion verification before ticket submission.
*   [ ] **Status Badge**: Clear "Open", "Replied", "Closed" badges on ticket list.
*   [ ] **Urgency Flag**: Allow users to mark ticket as "Urgent" (rate limited).
*   [ ] **Priority Queue**: "Pro users get answered within 1 hour" badge.
*   [ ] **Video Upload**: Allow small screen recording uploads for bug reports.
*   [ ] **Common Issues**: "Top 5 questions this week" dynamic list.
*   [ ] **Ticket Export**: Download ticket history as CSV.
*   [ ] **Reopen Ticket**: Allow reopening closed tickets within 3 days.
*   [ ] **Drafts**: Auto-save ticket draft while typing.
*   [ ] **Category Select**: Dropdown for "Billing", "Tech", "Sales".
*   [ ] **Related Project**: Dropdown to associate ticket with specific project ID.
*   [ ] **Response Time**: "Avg response time: 2 hours" indicator.
*   [ ] **CC Email**: Add additional emails to copy on the thread.
*   [ ] **Markdown Support**: Enable bold/code blocks in ticket body.
*   [ ] **Ticket Search**: Filter tickets by keyword.
*   [ ] **System Info**: Auto-append browser/OS info to new tickets.
*   [ ] **Closure Reason**: Dropdown for user to say "Solved", "Workaround", etc.
*   [ ] **Live Status**: "Agent is typing..." indicator.
*   [ ] **Feedback Text**: Optional comment field when closing ticket.
*   [ ] **Call Request**: "Request a callback" button (VIP only).
*   [ ] **Screenshare**: "Request screenshare" integration access.
*   [ ] **Knowledge Result**: "Is this article relevant?" inline result.
*   [ ] **Ticket Age**: "Opened 3 days ago" time ago string.
*   [ ] **Attachment Preview**: Image thumbnail instead of just filename.
*   [ ] **Bulk Archive**: Move old tickets to archive folder.
*   [ ] **Sort Order**: Toggle "Newest First" vs "Oldest First".
*   [ ] **Unread Dot**: Blue dot for tickets with new admin replies.
*   [ ] **Ticket ID**: "Copy ID" button for reference.
*   [ ] **Paste Image**: Allow CMD+V to paste image from clipboard.
*   [ ] **Reply Template**: "Use previous reply" for users (if they repeat themselves).
*   [ ] **Agent Profile**: "You are talking to [Name]" small avatar.
*   [ ] **Office Hours**: " We are currently CLOSED. Open in 8 hours."
*   [ ] **Ticket Limit**: "You have reached max open tickets" (spam protection).
*   [ ] **Language**: "I prefer support in [Spanish]".
*   [ ] **Print Thread**: Clean print view of conversation.
*   [ ] **Escalate**: "Request manager review" button (use sparingly).
*   [ ] **Satisfaction Trend**: "You rated last 3 tickets 5/5".
*   [ ] **Link Detection**: Auto-linkify URLs in messages.
*   [ ] **Availability**: "Expected wait time: 10 mins".
*   [ ] **Announcement**: "We are aware of the issue with X" banner in support.
*   [ ] **Bot Assist**: "AI Bot" first response toggle.
*   [ ] **Ticket History**: "You have 53 closed tickets" count.
*   [ ] **Privacy**: "Mark as sensitive" checkbox (hides from other team members).
*   [ ] **SLA Status**: "Guaranteed reply by 5:00 PM" badge.
*   [ ] **Phone Support**: "Pro #: +1-800-XXX-XXXX" display.
*   [ ] **Twitter Support**: Link to "@SupportHandle".
