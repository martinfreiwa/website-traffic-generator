# Affiliate Program Implementation Plan

**Component**: `Affiliate.tsx`
**Route**: `/dashboard/affiliate`

## Purpose
Allows users to earn credits or cash by referring others.

## Features
*   [ ] **Referral Link**: Unique link generation.
*   [ ] **Stats**: Clicks, Signups, Conversions.
*   [ ] **Payouts**: Request payout or convert to credits.

## Planned Improvements
*   [ ] **Marketing Assets**: Downloadable banners and social media kits (ZIP download).
*   [ ] **Leaderboard**: Top referrers rankings (gamification).
*   [ ] **Email Invites**: Send invitations directly from the dashboard.
*   [ ] **Custom Referral Code**: Allow users to customize their link (e.g., /ref/mybrand).
*   [ ] **Tiered Commission**: Visual progress bar to next commission percentage (e.g., "Refer 5 more to hit 15%").
*   [ ] **Payout Methods**: Selector for PayPal, Bank Transfer, or Crypto Wallet.
*   [ ] **Sub-Affiliates**: Earn percentage of referrals' referrals (MLM style).
*   [ ] **Conversion Pixel**: Instructions on where to place usage pixel.
*   [ ] **Social Share**: One-click "Share on X/Facebook" buttons.
*   [ ] **Commission Calculator**: "If I refer 10 people, I make $X".
*   [ ] **Link Shortener**: Built-in short URL for cleaner sharing.
*   [ ] **Campaign Tracker**: Append `?campaign=twitter` to track source performance.
*   [ ] **Payout History**: Table of all past payments with PDF receipts.
*   [ ] **Referral List**: Anonymized list of who signed up (e.g., "User ***45").
*   [ ] **Minimum Payout**: Progress bar specifically for "$50 minimum" threshold.
*   [ ] **Auto-Payout**: "Automatically pay me every month" toggle.
*   [ ] **TOS Agreement**: Mandatory "I agree to affiliate terms" checkbox.
*   [ ] **Notification**: Email alert on new signup.
*   [ ] **Bonus Challenges**: "Get 3 signups this week for $50 bonus" card.
*   [ ] **Partner Badge**: "Certified Partner" badge for high earners.
*   [ ] **Testimonial Submit**: "Tell us how you promote" form.
*   [ ] **Graph Clicks**: Line chart of clicks over last 30 days.
*   [ ] **Graph Earnings**: Bar chart of earnings per day.
*   [ ] **EPC Metric**: "Earnings Per Click" display.
*   [ ] **Conversion Rate**: "Signup %" metric.
*   [ ] **Landing Page**: Select which page the referral link points to (Home vs Pricing).
*   [ ] **QR Code**: Generate QR code for referral link.
*   [ ] **Deep Link**: Generator to link to specific blog posts with ref ID.
*   [ ] **Competitor Compare**: "Why we pay more than X" collateral one-pager.
*   [ ] **Success Stories**: "Read how user Y made $5k".
*   [ ] **Asset Preview**: Hover to see banner preview before download.
*   [ ] **Click geo**: Table of country origin of clicks.
*   [ ] **Device Stats**: "Most referrals from Mobile vs Desktop".
*   [ ] **Refunded Comm**: Highlight in red if a commission was clawed back.
*   [ ] **Tax Form**: "Upload W9" status indicator.
*   [ ] **Account Manager**: "Your contact is [Name]" (for VIPs).
*   [ ] **News Ticker**: "New banner ad sizes available now!".
*   [ ] **Integration Guide**: Link to "How to integrate API".
*   [ ] **Coupons**: "Create custom 10% off coupon".
*   [ ] **Postback URL**: Server-to-server tracking for pros.
*   [ ] **Fraud Warning**: "Self-referrals are not allowed" prominent note.
*   [ ] **Cookie Life**: "Cookies last 90 days" info badge.
*   [ ] **Payment Schedule**: "Next payout on [Date]".
*   [ ] **Referral Source**: "Came from Google / Direct".
*   [ ] **Pending Approval**: "Your application is under review" state.
*   [ ] **Whitelabel**: Host affiliate dash on your own domain.
*   [ ] **Multi-Currency**: Earn commissions in multiple currencies.
