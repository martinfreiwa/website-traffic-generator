# User Profile Implementation Plan

**Component**: `Profile.tsx`
**Route**: `/dashboard/profile`

## Purpose
Manages user account settings, security, and preferences.

## Features
*   [ ] **Personal Info**: Name, Email, Phone.
*   [ ] **Security**: Change password, 2FA toggle.
*   [ ] **API Access**: API Key management for developers.
*   [ ] **Notifications**: Email alert preferences.

## Planned Improvements
*   [ ] **Login History**: Table showing recent login IPs and devices.
*   [ ] **Avatar Upload**: Allow custom profile pictures.
*   [ ] **Data Export**: GDPR compliant full data download.
*   [ ] **Delete Account**: Self-serve account deletion flow (with confirmation).
*   [ ] **Timezone Setting**: Adjust all dashboard timestamps to user preference.
*   [ ] **Linked Accounts**: View and Manage OAuth connections (Google/Github).
*   [ ] **Social Links**: Add Discord/Twitter/LinkedIn to profile (for community badges).
*   [ ] **Developer Mode**: Toggle to show raw JSON responses in dashboard.
*   [ ] **API Key Regeneration**: One-click rotate secret key (with warning).
*   [ ] **Theme Customizer**: Pick specific accent color for keyboard.
*   [ ] **Language**: UI language preference (if multilingual).
*   [ ] **Email Frequency**: "Daily digest" vs "Instant" alerts.
*   [ ] **Public Profile**: Toggle to make profile public or private.
*   [ ] **Skills Badge**: "SEO Expert", "Traffic Master" badges based on usage.
*   [ ] **Newsletter Sub**: Toggle to subscribe/unsubscribe from marketing emails.
*   [ ] **Display Name**: "Show as [Nickname]" instead of real name.
*   [ ] **Recovery Email**: Secondary email for emergency account access.
*   [ ] **Session Kill**: "Log out everywhere else" button.
*   [ ] **Sms Alerts**: Toggle for critical alerts via SMS.
*   [ ] **Beta Tester**: Checkbox to opt-in to experimental features.
*   [ ] **Keyboard Shortcuts**: View list of all available shortcuts.
*   [ ] **Referral Code**: Custom field to set vanity referral code.
*   [ ] **Invoice Info**: Pre-fill Company Name/VAT for invoices.
*   [ ] **Onboarding Reset**: "Restart tour" button.
*   [ ] **Color Blind Mode**: Accessibility toggle for high contrast.
*   [ ] **Compact Mode**: "Reduce padding" toggle for denser UI.
*   [ ] **Font Size**: "Small / Medium / Large" text preference.
*   [ ] **API Whitelist**: Restrict API key usage to specific IPs.
*   [ ] **Sound Effects**: Toggle UI clicks/notifications sounds.
*   [ ] **Browser Notify**: Request permission for browser push notifications.
*   [ ] **Data Processing Agrmnt**: "Download DPA" pdf button.
*   [ ] **Risk Score**: (Internal) display user's own fraud score if transparent.
*   [ ] **Support Pin**: 4-digit pin to verify identity via phone support.
*   [ ] **Date Format**: DD/MM/YYYY vs MM/DD/YYYY toggle.
*   [ ] **Number Format**: 1.000,00 vs 1,000.00 toggle.
*   [ ] **Bio**: Short text bio for public profile.
*   [ ] **Job Title**: "SEO Manager at [Company]" field.
*   [ ] **Website**: Link to user's personal site.
*   [ ] **Team ID**: "Member of Team [Name]" indicator.
*   [ ] **Role**: "Admin / Editor / Viewer" role display (if enterprise).
*   [ ] **Audit Log**: "You changed password 3 days ago" timeline.
*   [ ] **API Logs**: link to "View API usage history".
*   [ ] **Webhook Secret**: View/Rotate webhook signing secret.
*   [ ] **Accessibility**: "Reduce Motion" preference sync.
*   [ ] **Marketing Consent**: Granular checkboxes for "Sales calls", "Updates".
*   [ ] **Password Age**: "Last changed 3 months ago".
*   [ ] **TOTP Reset**: QR code for setting up Authenticator app if lost.
*   [ ] **Emergency Codes**: Download list of one-time backup codes.
*   [ ] **Login Notification**: "Email me on new login" toggle.
